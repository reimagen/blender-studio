# Animation Brief

Use this before asking an AI agent to build animation shots.

## Project

- Title: Bubbles Rise - Micro Loop
- One-line concept: A calm macro-style bubble cluster rises through soft volumetric light with subtle camera drift.
- Target duration (seconds): 8
- Target fps: 24
- Target resolution: 1920x1080
- Aspect ratio: 16:9
- Style references: clean minimal CGI, soft gradients, dreamy depth of field, low visual noise

## Story and intent

- What should the viewer feel? Calm, floating, meditative motion with a premium polished look.
- Beginning: Establish cluster composition and gentle upward movement.
- Middle: Introduce slight camera parallax and bubble interaction overlap.
- End: Smooth settle into loop-safe framing for repeat playback.

## Technical constraints

- Hardware limits (GPU/CPU/RAM): Optimize for laptop workflow; keep viewport responsive.
- Render engine (Cycles/Eevee): Eevee for iteration, Cycles for final if needed.
- Max render time target: <= 30 minutes for final 8-second test export.
- File size limits: Keep preview renders under 200 MB.

## Assets

- Existing assets to use: `scenes/bubbles.blend` scene baseline.
- Assets to create: camera path, keyframes, lighting rig polish, optional subtle fog volume.
- External assets to import: none required for v1.

## Done criteria

- [ ] Camera motion approved
- [ ] Lighting approved
- [ ] Animation timing approved
- [ ] Test render approved
- [ ] Final render exported
